How does your city fare?

How does your city hold up in conversations about social issues, on the Twitter platform?

To start the application, press the play button, and wait for a prompt asking for a location.

This application takes in a single parameter (your location, in City/State format) and determines both the number of tweets that are likely socially insensitive (racist, sexist, etc.), and the number of tweets that likely express support for organizations that are devoted to bringing attention to fundamental social issues; this algorithm only covers the past 7-days of tweets in a particular region, due to limitations with the Tweepy API [1 -- Search Methods].

To rank the relative "sensitivity" of a tweet, the application relies on two sets of hashtags: one which denotes social insensitivity, and one that denotes social solidarity and understanding. I plan to consistently update these hashtags, as Twitter trends are certainly subject to change over time.

Note: if you are launching the site for the first time, the first use of the algorithm will be pretty slow, as the program host (Repl.it) needs to download relevant program packages in order for the program to function. If your browser stores cookies, subsequent launches of the application will be much faster!

This project makes use of Python libraries Tweepy (tweet analysis) [2] and GeoPy (converts a query in City/State format to GPS coordinates). [3]
